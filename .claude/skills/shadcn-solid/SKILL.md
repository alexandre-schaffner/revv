# shadcn-solid (v0.6.x)

Copy-paste component library for SolidJS built on Kobalte primitives + Tailwind CSS. Pre-styled, fully accessible, fully customizable. No installation—copy components into your project and modify as needed.

## Philosophy

- **Copy, not install.** Components live in your codebase; modify freely
- **Accessible by default.** Kobalte + ARIA best practices included
- **Unstyled foundation.** Kobalte primitives + Tailwind—fully themeable
- **Dark mode.** Built-in dark mode support via CSS classes
- **TypeScript.** All components fully typed

## Getting Started

1. Visit https://solid-ui.com
2. Browse components
3. Copy code into `src/components/ui/`
4. Customize colors, spacing, styles as needed
